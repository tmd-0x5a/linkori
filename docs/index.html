Organize everything.

Playlists, favorites, tags, and reading status — all in one sidebar.

Favorites & Tags

  • Star any playlist to mark as favorite
  • Filter the sidebar to ★ only with one click
  • Add custom text tags to each playlist
  • Click any #tag in the filter bar to narrow the list instantly

Reading Status & Progress

  • Mark playlists as Unread / Read from the right-click menu
  • Progress bar on each playlist shows how far you've read
  • Position is saved per playlist — resumes exactly where you left off

Always in the right order.

Drag, rename, or delete — right where you are.

Drag Anywhere to Reorder

No handle required. Grab any part of a playlist item and drag it to its new position.

Inline Rename

Double-click or press F2 to rename in place. No dialogs, no interruptions.

Right-Click Menu

Rename, delete, toggle favorite, edit tags, or change reading status — all from the context menu.

Chunks. Reading ranges, defined.

Link a start file to an end file. Chain chunks into a seamless reading session.

Flexible Range Definition

  • Point to a start image and an end image — everything in between is included
  • Set a folder or ZIP as start path to include the entire contents
  • Reverse order: end before start → images delivered in reverse
  • Optional name label for each chunk

Image Count & Preview

  • Image count badge on every chunk card
  • Click the eye icon to open a thumbnail gallery before launching the viewer
  • Right-click for edit and delete

Move them all at once.

Select multiple chunks, then drag. All of them move simultaneously — exactly where you want them.

Click to select

Single click selects one chunk.

Ctrl+click to add

Toggle individual chunks in or out of the selection.

Shift+click for range

Select a continuous range from the last selection to here.

Drag to select

Drag across the list to sweep-select multiple chunks.

Sort. Split. Undo.

Power tools for large playlists.

Sort Chunks

  • Name A→Z or Z→A
  • Modified date — newest or oldest first
  • Created date — newest or oldest first

Split by Subfolder

  • One chunk → one sub-chunk per subfolder
  • Nested subfolder support
  • Undo the split with a single click

Batch Delete & Undo

  • Select multiple chunks and delete them all at once
  • Ctrl+Z to instantly restore the last deleted batch

Browse without leaving the app.

A full file explorer — built in.

Navigation

  • Drive sidebar — all available drives listed automatically
  • Quick access — Desktop, Downloads, Documents, Pictures, Videos, Music
  • Address bar — breadcrumb navigation or type a path directly
  • Navigate into directories and ZIP / CBZ archives
  • Shift-JIS (CP932) filenames displayed correctly

Display & Selection

  • Search — filter visible files by filename
  • List / Grid view — toggle between list and large thumbnail grid
  • Sort — by name or date, asc/desc; remembered across navigations
  • Hidden files toggle — show or hide system/hidden files
  • Lazy thumbnail loading for fast browsing
  • "Select this folder" / "Select this ZIP" buttons
  • End-path locked to same folder/ZIP as start

A viewer built for manga.

Spread mode. RTL. Instant open. Zero waiting.

Display Modes

  • Single page or two-page spread — toggle with S key
  • Right-to-left reading direction (RTL) for Japanese manga
  • Page transition animation
  • Fullscreen (F key)

Performance

  • Instant open — viewer starts immediately; images load on demand as you read
  • Reading position memory — page and display mode saved per playlist across restarts
  • Smart seek-bar thumbnails — priority loading from current page; cached images resize instantly

Navigate with precision.

The page bar tells you exactly where you are.

Progress Bar

Drag or click the RTL progress bar to jump anywhere in the playlist instantly.

Chunk Tick Marks

Chunk boundaries appear as tick marks on the progress bar. Click any tick to jump directly to that chunk's first page.

Chunk Navigation

Prev/next chunk buttons flank the bar. Current chunk name shown for multi-chunk playlists.

Resilient. Reliable.

Keep reading even when something goes wrong.

Partial Failure Warning

If some chunks fail to load but others succeed, a dismissible banner at the top of the viewer lists the failed chunks. The rest of the playlist remains fully viewable.

PDF Support

Add PDF files as chunks. Pages are rendered on demand via pdf.js — no extraction, no conversion needed.

Multi-Select & Group Drag
