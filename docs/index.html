Everything you need to read.

Folders, ZIPs, PDFs, and playlists — all in one app.

Spread & Single View

Toggle between two-page spread and single-page mode. Right-to-left reading for manga.

Playlist System

Chain multiple folders and ZIPs into a single continuous reading session with chunks. Add tags, favorites, and reading status.

Built-in File Browser

Navigate drives, folders, and ZIP archives with search, sorting, thumbnails, and grid view.

Instant Open

Viewer opens immediately. Images load on demand as you read — no waiting, even for large ZIPs.

ZIP, CBZ & PDF

Read directly from ZIP archives (including subdirectories), CBZ files, and PDF documents. Shift-JIS filenames supported.

EN / JA Language

Switch UI language between English and Japanese. Preference is saved across sessions.

Feature deep dive.

Every feature, documented.

Playlist Management

  • Create, rename, and delete playlists
  • Drag anywhere on an item to reorder (no handle needed)
  • Double-click or F2 to rename in place
  • Right-click context menu: rename, delete, favorite, tags, status
  • Favorites — star playlists; filter bar shows ★ only
  • Custom tags — add text tags; click #tag in filter bar to narrow the list
  • Reading status — mark as Unread / Read from context menu
  • Reading progress bar on each playlist item

Chunk Management

  • Register start→end file range as a chunk (optional name)
  • Set a folder/ZIP as start path to include entire contents
  • Reverse order: end before start → images served in reverse
  • Multi-select — click, Ctrl+click, Shift+click, or drag to select
  • Group drag — all selected chunks move simultaneously via DragOverlay
  • Batch delete — delete selected; Ctrl+Z to undo
  • Sort — by name / modified date / created date (asc/desc)
  • Split by subfolder — expand one chunk into per-subfolder sub-chunks; undoable
  • Preview: eye icon → thumbnail gallery before opening viewer
  • Image count badge on every card

File Browser

  • Built-in explorer — no OS native dialog
  • Sidebar: all available drives detected automatically
  • Quick access — Desktop, Downloads, Documents, Pictures, Videos, Music
  • Address bar: breadcrumb navigation or type path directly
  • Search — filter visible files by name
  • List view / Grid view (large thumbnails) toggle
  • Sort: name or date, asc/desc — remembered across navigations
  • Hidden files toggle
  • Navigate into folders and ZIP / CBZ archives
  • Shift-JIS (CP932) filenames displayed correctly
  • Lazy thumbnail loading
  • "Select this folder" / "Select this ZIP" for whole-folder selection
  • End-path picker locked to same folder/ZIP as start path

Viewer

  • Single page / two-page spread toggle (S key)
  • Right-to-left reading direction (RTL, Japanese manga)
  • Page transition animation
  • Fullscreen (F key or button)
  • Instant open — lazy loading, no startup delay
  • Reading position memory — saves page & display mode per playlist across restarts
  • Seek-bar thumbnails — priority loading from current page; instant resize for cached images
  • Partial failure warning — banner lists failed chunks; rest of playlist still viewable
  • Chunk tick marks on progress bar — click to jump to chunk
  • Prev/next chunk buttons flank the progress bar
  • Current chunk name displayed when playlist has multiple chunks
  • PDF rendering via pdf.js (page-by-page, no extraction needed)

v0.4.0 Highlights.

Group drag & power selection.

New

Multi-Select & Group Drag

Ctrl+click, Shift+click, or drag to select multiple chunks. Drag the group — all selected chunks move simultaneously via DragOverlay with a stacked card and count badge.

New

Playlist Favorites & Tags

Star any playlist as a favorite and filter to ★ only. Add custom text tags to playlists and click a tag in the filter bar to narrow the list instantly.

New

Split Chunk by Subfolder

Split one chunk into multiple sub-chunks, one per subfolder — perfect for series organized by volume. Undo the split with a single click.

Improved

Playlist Drag Reorder

Drag anywhere on a playlist item to reorder — no dedicated handle required. Sort chunks by name, modified date, or created date from the sort menu.
