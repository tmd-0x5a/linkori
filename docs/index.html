Spread & Single View

Playlist System

Chain multiple folders and ZIPs into a seamless session. Tags, favorites, and reading status included.

Built-in File Browser

Search, thumbnails, list/grid, ZIP navigation — entirely inside the app.

Instant Open

Viewer starts immediately. Images load on demand as you read — no waiting.

ZIP, CBZ & PDF

Read directly from archives and PDF documents. Shift-JIS filenames supported.

EN / JA Language

Switch UI language between English and Japanese. Saved across sessions.

Organize everything.

Favorites, tags, and reading status — right in the sidebar.

Favorites

Star any playlist. Filter the sidebar to ★ only with one click.

Custom Tags

Add text tags to each playlist. Click any #tag in the filter bar to narrow the list instantly.

Reading Status & Progress

Mark as Unread / Read from the context menu. Progress bar on each item shows how far you've read.

Manage playlists your way.

Drag, rename, or act — right where you are.

Drag Anywhere to Reorder

No handle required. Grab any part of a playlist item and drop it into position.

Inline Rename

Double-click or press F2 to rename in place. No dialogs.

Right-Click Menu

Rename, delete, favorite, edit tags, or change reading status — all from the context menu.

Chunks. Reading ranges, defined.

Link a start file to an end file. Chain chunks into one seamless read.

Flexible Range

  • Start image → end image — everything between is included
  • Set a folder or ZIP as start to include entire contents
  • Reverse order: end before start → images in reverse
  • Optional name label per chunk

Count Badge & Preview

  • Image count badge displayed on every chunk card
  • Eye icon → thumbnail gallery before opening the viewer
  • Right-click for edit and delete

Move them all at once.

Select multiple chunks, then drag. All of them move simultaneously — exactly where you want them.

Click

Select a single chunk.

Ctrl+Click

Toggle chunks in or out of the selection.

Shift+Click

Select a continuous range.

Drag

Sweep across the list to select a range.

Sort. Split. Undo.

Power tools for large playlists.

Sort Chunks

  • Name A→Z or Z→A
  • Modified date — newest or oldest first
  • Created date — newest or oldest first

Split by Subfolder

  • One chunk → one sub-chunk per subfolder
  • Nested subfolder support
  • Undo the split with a single click

Batch Delete & Undo

  • Select multiple chunks and delete them all at once
  • Ctrl+Z to restore the last deleted batch instantly

Browse without leaving the app.

A full file explorer, built in.

Navigation

  • Drive sidebar — all available drives listed automatically
  • Quick access — Desktop, Downloads, Documents, Pictures, Videos, Music
  • Address bar — breadcrumb or type a path directly
  • Navigate into folders and ZIP / CBZ archives
  • Shift-JIS (CP932) filenames displayed correctly

Search & Display

  • Search — filter visible files by filename
  • List / Grid view — large thumbnail grid or detailed list
  • Sort — name or date, asc/desc; remembered across navigations
  • Hidden files — show or hide with one click
  • "Select this folder" / "Select this ZIP"
  • End-path locked to the same folder/ZIP as start

  • Right-to-left (RTL) for Japanese manga

Performance

Navigate with precision.

The page bar tells you exactly where you are.

Progress Bar

Drag or click the RTL progress bar to jump anywhere instantly.

Chunk Tick Marks

Chunk boundaries appear on the progress bar. Click any tick to jump to that chunk.

Chunk Navigation

Prev/next chunk buttons flank the bar. Current chunk name shown for multi-chunk playlists.

Resilient. Reliable.

Keep reading even when something goes wrong.

Partial Failure Warning

If some chunks fail but others succeed, a dismissible banner lists the failed chunks. The rest remains fully viewable.

PDF Support

Add PDF files as chunks. Pages rendered on demand via pdf.js — no extraction needed.
